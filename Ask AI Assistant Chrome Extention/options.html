Having trouble connecting?

Ollama (Connection Refused)

By default, Ollama blocks browser extensions. You must allow Cross-Origin Requests (CORS).

  • Linux:
    Run
    sudo systemctl edit ollama.service
    and add
    Environment="OLLAMA_ORIGINS=*"
    under [Service].
    Restart with
    sudo systemctl restart ollama.
  • Mac/Windows:
    Open your terminal/command prompt
    and run set OLLAMA_ORIGINS="*" (Windows)
    or export OLLAMA_ORIGINS="*" (Mac),
    then start the Ollama app
    from that same terminal.

Ollama (Model Not Found)

If you get an error when chatting, ensure you have downloaded the exact model name. Open your terminal and run: ollama run <model-name>.

Cloud APIs (Groq, OpenAI, Gemini)

If the chat fails to load:

  • Double-check that your API key is pasted correctly without extra spaces.
  • Ensure your API account has available credits (some free tiers expire).
  • If using a custom model, ensure the string exactly matches the provider's documentation.
